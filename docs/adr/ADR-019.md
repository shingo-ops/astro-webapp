# ADR-019: Meta審査期間中の英語UI一時デプロイ

## Status
Proposed — 2026-05-09

## What
Meta App Review 期間中のみ、`recording/english-ui` ブランチを
本番環境（app.salesanchor.jp）に一時デプロイする。
審査通過後は develop ブランチの日本語 UI に戻す。

## Why
Meta のレビュアーはスクリーンキャストだけでなくアプリを LIVE で
操作して Permission の動作を確認する。
本番アプリが日本語 UI のままだとレビュアーが操作できず
リジェクトリスクが高まる。

## Scope外
- i18n 基盤の整備（react-i18next 等の導入）
- 永続的な英語 UI 化
- バックエンド API のレスポンス英語化

## パートナーへの委任事項
1. `recording/english-ui` を main にマージしてデプロイする手順の実装
2. 審査通過後に develop を main に再マージして日本語 UI に戻す手順の整備
3. `deploy.yml` への影響確認（migration046 等の冪等性確認）

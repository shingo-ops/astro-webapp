# ADR-020: recording/english-ui 本番デプロイ実行

## Status
Proposed — 2026-05-09

## What
`recording/english-ui` ブランチを main にマージして
本番環境（app.salesanchor.jp）に英語 UI をデプロイする。

## Why
Meta App Review のレビュアーがアプリを実際に操作して
Permission の動作確認を行うため、本番環境が英語 UI である必要がある。
ADR-019 で手順書は整備済み。

## Scope外
- 新機能の追加
- バックエンドの変更
- 日本語 UI への復帰（審査通過後に別途実施）

## パートナーへの委任事項
ADR-019 runbook §2 の手順通りに実行する：
1. `recording/english-ui` を最新 main に追従（rebase）
2. `recording/english-ui → main` の PR を作成
3. `deploy.yml` が自動実行されて本番に英語 UI が反映される
4. `app.salesanchor.jp` のヘルスチェック確認

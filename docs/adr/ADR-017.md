# ADR-017: robots.txt に facebookexternalhit を許可

## Status
Proposed — 2026-05-08

## What
salesanchor.jp の robots.txt に以下を追加する：

```
User-agent: facebookexternalhit
Allow: /
```

## Why
Meta Business Verification のドメイン認証において、
Meta のクローラー（facebookexternalhit）が salesanchor.jp を
スクレイピングしようとしたところ 403 でブロックされていることが
Sharing Debugger Tool で判明した。
この設定を追加しないとドメイン認証が完了できない。

## Scope外
- 他のクローラーへの影響（既存の robots.txt 設定は変更しない）
- SEO への影響（facebookexternalhit は SEO クローラーではない）

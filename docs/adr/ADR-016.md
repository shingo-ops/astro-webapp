# ADR-016: Facebook ドメイン認証メタタグの追加

## Status
Proposed — 2026-05-08

## What
salesanchor.jp のフロントエンドHTMLの <head> セクションに
Facebook ドメイン認証用のメタタグを1行追加する。

```html
<meta name="facebook-domain-verification" content="r38od7qknawjulwqpmvbpso1q1zfy4" />
```

## Why
Meta Business Verification（ビジネス認証）の申請プロセスにおいて、
salesanchor.jp のドメイン所有権をMetaに証明するために必要。
認証完了後、Meta App Reviewの申請が進められるようになる。

## Scope外
- その他のHTMLへの変更
- メタタグの削除（認証完了後も残す）
